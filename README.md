# Gusto Restaurant Website

A fully responsive restaurant website with menu display, online booking system, and multiple pages.

## Features

- **Responsive Design**: Works on all devices (mobile, tablet, desktop)
- **Online Booking System**: Form for table reservations
- **Menu Display**: Showcases dishes with prices and descriptions
- **Multiple Pages**: Home, About, Services, Contact
- **Interactive Elements**: Smooth scrolling, form validation
- **Modern UI**: Clean and elegant design

## Pages Structure

1. **index.html** - Homepage with hero section, menu highlights, and booking form
2. **about.html** - Restaurant story, team information
3. **services.html** - Services offered and full menu
4. **contact.html** - Contact information and message form
